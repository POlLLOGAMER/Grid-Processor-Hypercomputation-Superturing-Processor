# GDSII Output Directory

This directory contains the generated GDSII files after OpenLane 2 and GDS merge scripts complete.

## Expected Output Files

| File | Description |
|------|-------------|
| `tt_um_grid_oracle.gds` | Raw OpenLane 2 output (before seal ring) |
| `tt_um_grid_oracle_sealed.gds` | With SkyWater 130nm seal ring added |
| `tt_um_grid_oracle_filled.gds` | Final DRC-clean GDS with fill cells |
| `drc_report.xml` | KLayout DRC verification report |
| `netlist.spice` | Extracted SPICE netlist (for LVS) |

## Generation Pipeline

```
OpenLane 2 PnR → Raw GDS → KLayout Seal Ring → KLayout Fillers → DRC Check → Final GDS
```

These files are generated by running:
```bash
python3 scripts/build_grid_sky130.py --output-dir .
bash scripts/generate_gds.sh
```
